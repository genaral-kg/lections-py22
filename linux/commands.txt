CTRL+ALT+T - команда открытия терминала
MacOs(В спотлайт вводим Terminal)

~ - символ домашней дериктории(папки)
/ - символ корневой директории(папки)


.. - обозначает предыдущую дерикторию
. - обозначает текущию директорию 

1. ls (list) -> Выводит список всех директорий и файлов в текущей директории(рабочая директория)


ls  -  a - выводить список как и скрытоых так и обвчных диреткории

ls - l - выодит список директорий и файлов с подробной информацией



2.cd(change directory) - менят текущию директорию на укзанную
cd<название директории>

3.mkdir (make directory) -  создает указанную директорию 
mkdir<дирекория>

4.touch - создает указанный файлte
touch<name_of_file>.<extension(расширение (txt,docs))>

5.pwd(print workind directory) - выводит путь до
текущей дирекорииe

6.rmdir ( remove directory)  - удаляет указанную директорию ( не может удалить не пустую директорию)

7. rm -> удаление файла
rm<name_of_file>


rm -rf -> принудительное удаление файла либо директории

8. mv (move) ->  перермещает либо переименовывает файл либо дирекорию, в зависимости от указанного второго аргумента


mv  <файл_директория><новое_имя--директория/>

9. cat -> выводит содердимое указанного файла 

10. nano -> открывает графический интерфейс в терминале для  редактирование файла
*если нет указанного  файла то создаться новый файл с таким именем

nano<name_of_file>



11. echo -> команда для быстрой записи данных в файл через терминал

echo  текст для записи > название файла 



12.cp (copy) - команда для копирование файла в нужную директорию


cp<file_to_copy><path>
